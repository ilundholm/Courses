# Courses: Interactive Math Demos

Welcome to the central repository for Dr. Lundholm's course demonstrations. This repository contains a collection of interactive web-based tools and visualizations designed to enhance the learning experience in various mathematics courses.

## Contents

- **Course Hub**: Access the central navigation page at `index.html`.
- **Course-Specific Demos**: Each course has its own directory (e.g., `351/`) containing relevant demos and templates.

## Usage

These demos are designed to be run in a modern web browser. Simply open the relevant `.html` file to view the interactive content.
